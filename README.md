> [!NOTE]
> ## PowerShell File Manager
> Interactive file management tool for PowerShell syntax practice. For learning/testing only.


> [!TIP]
> ## Usage
> Run `.\file-manager.ps1` and choose from:
> - **1** - Change Directory
> - **2** - Go Back  
> - **3** - Move File/Folder
> - **4** - Copy File/Folder
> - **5** - Delete File/Folder
> - **6** - Rename File/Folder
> - **7** - Create Folder
> - **8** - Exit

> [!IMPORTANT]
> ## Requirements
> PowerShell 5.1+
